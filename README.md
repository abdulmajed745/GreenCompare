GreenCompare - Plant Care Comparison Tool 🌱

Overview

GreenCompare is a Java application that helps plant enthusiasts compare care requirements between different plants. The system imports plant data from CSV into a MySQL database and provides a console interface for searching and comparing plant care information.

Features ✨

CSV to MySQL Import: Efficiently imports plant data from CSV files.

Plant Search: Find detailed care information for individual plants.

Plant Comparison: Side-by-side comparison of care requirements between two plants.

Threaded Operations: Uses multi-threading for responsive searches.

MySQL Integration: Stores and retrieves plant data from a relational database.
